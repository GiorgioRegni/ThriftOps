# Import Mapping

## Supported Inventory Workbook Sheets

- `Women`
- `Women - Sold`
- `Kids`
- `Kids - SOLD`
- `Home goods`
- `Home goods - Sold`

Active sheets create active items. Sold sheets create sold items. Item IDs are generated by category prefix and year.

## Supported Inventory Columns

- `Brand`
- `what?`
- `size`
- `cost`
- `Sold`
- `Shipping`
- `Benefice net`
- `Event / Market price`
- `Link`

Unknown columns are preserved in import metadata.

## Supported Accounting Workbook Sheets

- `2026 - Revenus`
- `2026 - expenses`
- `2025`
- `info`

Revenue rows become payment/import records where possible. Expense rows become expense records where possible. Unknown rows are staged for review.

## Review Flags

- `missing_cost`
- `missing_sale_price`
- `ambiguous_date_or_event`
- `missing_brand`
- `unknown_sheet`

## Limitations

Date and event inference is conservative. Ambiguous rows are flagged and original row data is preserved for review before writing.
